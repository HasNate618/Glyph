# Code of Conduct

This project follows the Contributor Covenant code of conduct. By participating, you agree to abide by its terms.

Summary
- Be respectful and welcoming.
- Be considerate of others and their time.
- Avoid harassment, discrimination, or personal attacks.

Reporting
- If you experience or witness unacceptable behavior, contact the project maintainers privately. Reports will be handled promptly and with discretion.

Full text
For the full text of the Contributor Covenant, version 2.1, see:
