# Glyph

Glyph is a Windows leader-key, layered key sequence engine with a discoverable overlay.

## Prerequisites
- Windows 10/11
- .NET SDK (8+)

## Build
`dotnet build Glyph.sln`

## Run
`dotnet run --project src/Glyph.App/Glyph.App.csproj`
